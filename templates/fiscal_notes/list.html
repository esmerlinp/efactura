Guía de Uso: Notas Fiscales

Gestiona las Notas de Crédito (E34) y Notas de Débito (E33) para realizar ajustes fiscales ante la DGII.

  • Nota de Crédito (E34): Para anular o reducir el valor de una factura ya emitida.
  • Nota de Débito (E33): Para incrementar el valor de una factura ya emitida o corregir montos.
  • Aprobación DGII: Una vez emitidas, las notas fiscales deben ser aprobadas por la DGII para tener validez fiscal.
