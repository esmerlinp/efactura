Catálogo de Productos

Administra los productos y servicios que ofreces, con precios, tasas de impuestos y control de inventario integrado.

Registro de Ítems

Ítem

Unidad básica del catálogo. Puede ser un producto físico, un servicio, un paquete o un cargo adicional. Cada ítem tiene nombre, descripción, precio y tasa de ITBIS.

SKU

Código único de identificación del ítem (Stock Keeping Unit). Facilita la búsqueda, el escaneo en POS y la integración con inventario.

Unidad de Medida

Define cómo se vende el ítem: Unidad, Lote, Caja, Metro, Hora, Servicio, etc. Afecta el cálculo de cantidades en facturas y órdenes de compra.

¿Cómo registrar un nuevo producto o servicio?

Accede al módulo Catálogo → Ítems (/items) y haz clic en Nuevo Ítem. Completa los siguientes campos:

Información básica:
  • Nombre del Ítem: El nombre comercial del producto o servicio (ej. "Consultoría Fiscal Mensual").
  • SKU: Código interno único. Si no lo especificas, el sistema puede generarlo automáticamente.
  • Descripción: Texto descriptivo que aparecerá en las facturas. Incluye detalles relevantes para el cliente.
  • Unidad de Medida: Selecciona entre Unidad, Servicio, Hora, Caja, Lote, Kg, etc.
Precio e impuestos:
  • Precio Unitario: Valor base del ítem sin impuestos (precio neto).
  • Tasa de ITBIS: Selecciona entre 18% (estándar), 0% (exento) o 16% (reducido para ciertos productos).
  • ITBIS Incluido: Si activas esta opción, el precio unitario se interpreta como precio final con ITBIS incluido y el sistema calcula el precio neto automáticamente.
Tipo de ítem:
  • Producto: Bien tangible con control de inventario (stock).
  • Servicio: Prestación intangible sin seguimiento de inventario.
  • Paquete: Combinación de varios productos o servicios que se venden como un conjunto.
  • Cargo: Costo adicional como flete, seguro, manejo, etc.
Tasas de ITBIS

ITBIS 18% (Estándar)

Tasa general aplicable a la mayoría de productos y servicios. Corresponde al ITBIS estándar de la legislación dominicana.

ITBIS 0% (Exento)

Productos o servicios exentos por ley: alimentos de canasta básica, servicios educativos, médicos, alquiler de viviendas, exportaciones, etc.

ITBIS 16% (Reducido)

Tasa reducida aplicable a ciertos bienes y servicios específicos según la legislación fiscal vigente.

¿Cómo seleccionar la tasa de ITBIS correcta?

La tasa de ITBIS que selecciones para cada ítem determina cómo se calcularán los impuestos en las facturas. Es importante elegir la tasa correcta:

18% (Estándar):
  • Aplica a servicios profesionales (consultoría, desarrollo de software, diseño, abogacía).
  • Aplica a productos manufacturados, electrónicos, ropa, muebles, etc.
  • Es la tasa por defecto al crear un nuevo ítem.
0% (Exento):
  • Productos de la canasta básica (arroz, habichuelas, aceite, leche, pan, etc.).
  • Servicios educativos (colegios, universidades, cursos formales).
  • Servicios médicos y hospitalarios.
  • Alquiler de viviendas para uso residencial.
  • Exportaciones de bienes y servicios.
  • Libros, periódicos y revistas.
16% (Reducido):
  • Aplica a productos específicos definidos por la legislación, como ciertos alimentos procesados.
  • Verifica con tu contador si tus productos califican para esta tasa.
Seleccionar una tasa de ITBIS incorrecta puede generar inconsistencias en tus declaraciones fiscales. Consulta con tu contador para determinar la tasa correcta para cada producto o servicio.
Categorías y Tipos

Organiza tu catálogo con categorías

Las categorías te permiten agrupar tus ítems para facilitar la búsqueda, los reportes y el análisis de ventas:

Crear categorías (/items/categories):
  • Accede a Catálogo → Categorías y haz clic en Nueva Categoría.
  • Define un Nombre (ej. "Electrónicos", "Servicios Profesionales", "Materia Prima").
  • Opcionalmente asigna un Color para identificarla visualmente en listas y gráficos.
  • Las categorías pueden organizarse en jerarquías (categoría padre → subcategoría).
Tipos de productos: Además de las categorías, cada ítem tiene un tipo que define su comportamiento:
  • Producto: Control de inventario habilitado. Puedes asignar almacén, stock mínimo y costo unitario.
  • Servicio: Sin control de inventario. Ideal para consultorías, honorarios, suscripciones.
  • Paquete: Agrupación de varios ítems. Al facturar un paquete, puedes desglosar o no sus componentes.
  • Cargo: Para costos adicionales como flete, instalación o seguro.
Listas de Precio

¿Cómo funcionan las listas de precio?

Las listas de precio te permiten definir diferentes precios para un mismo ítem según el cliente o canal de venta:

Crear una lista de precio:
  • Accede a Catálogo → Listas de Precio (/items/price-lists).
  • Haz clic en Nueva Lista y asígnale un nombre (ej. "Mayorista", "Cliente VIP", "Precio Regular").
  • Selecciona la Moneda de la lista (DOP, USD, EUR).
  • Define si la lista tiene un % de descuento o recargo sobre el precio base del ítem, o si defines precios específicos para cada ítem.
Asignar lista a clientes:
  • Desde el perfil del cliente (/clients), puedes asignarle una lista de precio específica.
  • Al crear una factura para ese cliente, el sistema cargará automáticamente los precios de su lista asignada.
  • Si un cliente no tiene lista asignada, se usa el precio base del ítem.
Los precios en las listas se expresan como precio sin ITBIS (precio neto). El sistema calcula el ITBIS al momento de facturar según la tasa configurada en el ítem.
Vinculación con Inventario

Configuración de inventario para productos

Para los ítems de tipo Producto, puedes habilitar el control de inventario desde el formulario de registro:

Stock inicial y almacén:
  • Selecciona el Almacén (Warehouse) donde se encuentra el producto. Si tienes múltiples almacenes, puedes definir stock por cada uno.
  • Ingresa la Cantidad Inicial de unidades disponibles.
  • Define un Stock Mínimo: cuando las existencias bajen de este número, el sistema activará una alerta visual.
Costo unitario y costeo:
  • El Costo Unitario es el valor que pagaste por adquirir o producir el producto. Se usa para calcular el margen de ganancia en los reportes de BI.
  • El método de costeo (FIFO, promedio ponderado o estándar) se configura desde el módulo de Inventario y aplica a nivel de empresa.
Comportamiento en facturación:
  • Cuando emites una factura que incluye un producto con control de inventario, el sistema descuenta automáticamente las unidades vendidas del stock del almacén correspondiente.
  • Si el stock es insuficiente, el sistema te advertirá antes de emitir la factura.
  • Las notas de crédito devuelven las unidades al inventario automáticamente.
